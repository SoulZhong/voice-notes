# 离线清洗神经残余级（DTLN-aec）— 设计

日期：2026-07-14
状态：已确认（P3b；叠栈于二期分支，依赖 echo_clean 管线）

## 背景与根因

一期离线清洗（AEC3 线性消除）把 mic 轨残余互相关从 >0.30 压到 ~0.13——线性
滤波器只建模 ~52ms 回声尾部，残余只能靠非线性抑制器掩蔽，这是「指标好但听感
仍有残影」的结构性根源（2026-07-14 深度调研 3-0 验证）。P3a 回放门控在回放侧
掩蔽了混音重影；本设计在**数据侧**把残余真正消掉。

选型依据（调研 + 真实录音 spike 实测）：
- DTLN-aec：MIT 许可含权重、TF-Lite、16kHz（与落盘格式零转换）、mic+loopback
  双信号输入与双轨一一对应、微软 AEC 挑战赛盲测残余回声抑制第三名、对延迟
  自适应无需预对齐（漂移场顺带受益）。
- **Spike 实测**（真实笔记重灾 300s 切片 + 本人独讲 120s 切片，256 档）：
  残余互相关 0.289 → 0.116；回声区包络再降 ~30dB；**本人独讲 19 段共 102s
  能量保持 -0.01dB（近端几乎无损,调研警告的近端衰减未在该模型+该数据上发生）**；
  python 下即 40 倍实时。
- 业界共识的混合架构（3-0 验证）：线性 AEC 在前、神经模型做后级——不全盘替换,
  避免端到端神经 AEC 的近端衰减风险。

## 用户决策（约束）

- 推理路线：~~tract-tflite 优先、tflitec 兜底~~ **两条原路线经可行性硬闸实测
  双双失败**（tract 的 tflite 导入器 ReduceMean 负轴越界 panic,最新版未修;
  tflitec 无预编译库需 bazel 从源码编 TF,不适合桌面依赖）。**改走第三路线并已
  spike 验证到底（用户批准）**：tflite→ONNX 离线转换一次（tf2onnx 1.17.0/
  opset 13,数值最大差 ~1e-6）+ **tract-onnx 纯 Rust**（0.21,rustc 1.89 兼容,
  双模型加载运行实证）。转换后工件挂本仓 release `models-dtln-aec-v1`,注册表
  指向之,release 说明含来源/转换方式/哈希/MIT 许可。
- 零用户配置：模型在场即启用，不在场静默跳过；无设置项。

## 架构

```
echo_clean::clean_wav(AEC3 pass 完成,Ok(Some) 路径)
  → [新] 神经残余级(增值层,失败保留 AEC3 输出):
      audio/neural_aec.rs::suppress_residual(mic_aec3, system_aligned, models_dir)
        - tract-tflite 加载 dtln_aec_256_1/2.tflite(共 ~15.5MB)
        - 官方块参数:512 块长 / 128 移位(8ms 步进)双阶段推理
        - LSTM 状态为显式输入/输出张量,逐块循环回喂(无内部状态,正适合 tract)
        - 样本守恒(输出 len == 输入 len,块尾余量原样)
  → 替换清洗输出 → 原子写 tmp → rename(既有流程不变)
```

- 参考信号直接用 `system_aligned`（预对齐后的参考;spike 用未对齐参考也有效,
  对齐后只会更容易）。
- 触发条件：AEC3 清洗实际执行（过双门限）才追加神经级;未检出回声的笔记不动。
- 模型缺失/加载失败/推理失败 → 保留 AEC3 输出照常落盘 + eprintln 一行。

## 错误处理

增值层三连:模型不在场→跳过;加载/推理失败→保留 AEC3 输出;任何路径不 panic、
不阻塞转码。神经级失败不回滚 AEC3 清洗成果。

## 测试与验收

- 可行性硬闸（首任务）：tract-tflite 加载双模型、对 1s 合成音频出数——失败即
  切 tflitec 并记录，后续任务不变。
- 合成单测：回声衰减（远端延迟拷贝 → 输出能量显著低于输入）;样本守恒;
  参考全零时近端直通（不被误削）。
- 真实录音验收（spike 同口径,同切片）：残余互相关 ≤0.15（spike 0.116）;
  本人独讲能量保持 |Δ|≤0.5dB;吞吐 ≥10 倍实时（Rust 应远超 python 的 40x）。
- Rust 输出与 python 参考实现同输入对比：统计等效断言（残余互相关差 <0.02),
  不做位级对比（跨运行时浮点差异）。
- 回归：全量绿。

## 非目标

- 不做实时神经级（离线专用;实时链路已有二期预对齐）。
- 不动 128/512 档（256 档 spike 已验证;将来要调再说）。
- 不引入 DeepFilterNet（后级降噪另立;本期只治残余回声）。
