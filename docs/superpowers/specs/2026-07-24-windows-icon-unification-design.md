# Windows 图标统一设计

## 目标

让 Voice Notes 在 Windows 任务栏中充分利用系统图标空间，并确保窗口、任务栏、
安装包、Windows 商店磁贴、网页 favicon 与系统托盘均使用当前最新版品牌头像。

## 母版与尺寸策略

- `src-tauri/icons/icon.png` 是唯一品牌母版。
- 将现有约 9.5% 的透明外边距缩减为约 3%，保持圆角背景、人物比例和构图不变。
- 从同一处理后的母版生成 `32x32.png`、`128x128.png`、`128x128@2x.png`、
  `icon.ico`、`icon.icns`、全部 Windows Square/Store 图标和 `static/favicon.png`。
- 小尺寸使用高质量缩放，并逐个检查透明边界，避免裁切头发、圆角或底部笔记本。

## 托盘图标

- 删除旧版卡通人物视觉，改用最新版头像。
- 保留现有录音动画和状态切换机制。
- 空闲帧与录音帧共享同一新版头像母版；录音帧只叠加状态动画，不再引入另一套人物。
- 托盘画布继续使用 44×44，主体尽量占满安全区域。

## 一致性检查

- 所有系统 PNG 必须能追溯到同一母版，不再保留旧人物资产。
- Tauri 配置继续引用标准图标文件名，避免改变打包接口。
- ICO 必须包含 Windows 常用的多分辨率层级，防止任务栏缩放时取错低清图层。
- 安装包生成后检查安装目录中的 EXE 图标、运行中的任务栏图标和托盘图标。

## 验证

1. 自动检查所有 PNG 的尺寸、透明边界和母版一致性。
2. 运行前端测试、Svelte 检查与 Windows Release 构建。
3. 静默安装并启动新版。
4. 冒烟测试：进程启动、主窗口可见、任务栏图标、托盘空闲图标、录音状态图标。
